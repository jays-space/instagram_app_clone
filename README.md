# Instagram Clone

Full-stack social media application replicating core Instagram features across web and mobile.

## Tech Stack

- **Web:** React
- **Mobile:** React Native
- **Backend:** AWS Amplify
- **Database:** DynamoDB
- **Storage:** S3
- **Auth:** Cognito

## Features

- User authentication and profiles
- Photo upload and sharing
- Like and comment system
- Follow/unfollow functionality
- Real-time feed updates
- Story-like ephemeral content
- Direct messaging
- Explore page with discovery algorithm

## Architecture

- Serverless backend with AWS Lambda
- GraphQL API (AppSync)
- Image optimization pipeline
- Push notifications

---

Built to explore complex state management, real-time features, and AWS serverless patterns.
